# Recorner
Java. Object recognizer [click to see using](https://youtu.be/k0LbaRUBKho)

![](https://github.com/StanislavBaranuk/Recorner/blob/master/logo.png)

### ABOUT

The program allows the user to apply one of the following functions:
1. Character recognition using the lower unit is output.
2. Clear the character bar.o8DwxWTac
3. Opening a photo from a computer.
4. Open the window to output the database:
a. View Characters in the Database.
b. View the total number of characters in the database.
c. Find the character you want in the database.
d View the number of instances of each character.
5. View the recognized characters.
6. Enter the character in the selected color (black / white).
7. Change the input color.
8. Increase the entered character.
9. Character recognition using the "IDENTIFY" button.
10. Training program.
11. Open the configuration window:
a. Setting the thickness when recognizing with a button.
b. Select a database.
c. Change the administrator password.
12. Automatic photo recognition.

> Before using make sure that file `chars.txt` is present in `./data/`

![](http://stanislas.zzz.com.ua/r-instr.png)

### HOW IT WORK

![](http://stanislas.zzz.com.ua/algo.jpg)

`User input` -> `Making photo from input` -> `Request to DB` -> `Get 5 the most similar objects from DB` -> `Show it to user` -> `Choose needed object`

### Object in DB

```
  { name: <char-code> value: white-black code 30 x 30 }
```
